Java日志库扫盲

1. 基本概念

slf4j：simple logging facade for Java，可以看作用来管理各个日志库的调度器，我们只需要通过配置slf4j后端具体的使用的日志库就可以完成日志操作，对用户隐藏了后端真正使用的日志库
log4j：一个日志库
Logback：日志库
